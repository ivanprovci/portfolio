Projects

KPU Scheduler

For our capstone project, my team and I developed a web application for the IT department head at KPU to streamline semester schedule creation. Our product automates various steps, significantly speeding up the process while performing conflict checks to eliminate errors—a major improvement over the school's legacy system. I was responsible for building the backend, which involved routing with Express.js, implementing authentication with cookies and JWT, and managing the database connection and queries using Pocketbase. I was also in charge of the front-end where I implemented numerous CRUD operations, and ensured the UI was intuitive and interactive by utilizing JS and Bootstrap.

Pixl Sketch

Pixl sketch is a drawing web app made with JavaScript and CSS as a submission for The Odin Project. This solo project was meant to practice dynamic JavaScript and DOM manipulation. The basic features include drawing, erasing, colour selecting, and resetting the grid to default. The fancier tools include pixel shading, pixel tinting, and a rainbow gradient pen!

Obsidian

Obsidian is a full-stack ecommerce web application that supports account registration, catalog filters, shopping cart functionality, and an account-based favourites section. The backend is built with PHP, and the front-end utilized Bootstrap for several components, and CSS for everything else. Account credentials were hashed and stored in a MySQL database, and can even be edited using the edit profile dropdown menu. The whole application is hosted using XAMPP's Apache and MySQL servers, which can be set up quickly by following the instructions in the README of the repository.

PHP MySQL Bootstrap

BC Health & Wellness

BC Health & Wellness is a feature-rich medical clinic system built with a team of 5 for a 'System Development Project' course. It supports patient, doctor, and staff accounts, each with their own account privileges. Patients can view their medical records, prescriptions, and lab exam results. Doctors can create new records, and the staff manage and approve accounts, without having access to confidential documents. I integrated the Spring Boot backend with the front-end by creating RESTful JavaScript API calls to our MySQL database. I plan on refactoring this project with Tailwind to polish up the mistakes we made back then.
